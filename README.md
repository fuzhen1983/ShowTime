# ShowTime

一款专为 Windows 打造的**极简全屏时钟**与**番茄计时器**。

无论你是需要专注工作的程序员、追求沉浸感的设计师，还是在课堂上需要清晰展示时间的讲师，ShowTime 都能让你的桌面时间既美观又实用。

![ShowTime Screenshot](https://showtime.jsonee.com/images/showtime-1.png) 

## ✨ 功能特性

- **🖥️ 全屏沉浸**：一键进入全屏模式，巨幅数字时钟，演讲授课神器。
- **🍅 番茄工作法**：内置 25 分钟专注 + 5 分钟休息计时，科学管理时间。
- **🎨 高度自定义**：支持自定义字体、颜色、背景透明度及背景图片，完美融入你的桌面美化主题。
- **🚀 轻量纯净**：基于 .NET 8 开发，体积小巧，无广告，不扰民。

> ⚠️ **关于 Windows 安全提示（重要）**
>
> 由于 ShowTime 是个人独立开发项目，尚未购买昂贵的商业代码签名证书，Windows SmartScreen 在首次运行从 GitHub 下载的文件时可能会弹出红色警告：
>
> **“Windows 已保护你的电脑”**
>
> **这不是病毒！** 解决方法很简单：
> 1. 点击弹窗中的 **“更多信息”**。
> 2. 此时会出现 **“仍要运行”** 按钮，点击即可。
>
> 首次通过后，后续使用将不再拦截。感谢您的理解与信任！🙏

## 🛠️ 使用说明

1. 启动 ShowTime。
2. 点击界面上的 **“全屏”** 按钮（或按 `F11`）进入沉浸式时钟模式。
3. 切换至 **“番茄钟”** 标签页，设置专注时长并开始计时。
4. 在 **“设置”** 中调整外观，打造你的专属风格。
